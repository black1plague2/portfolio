# Brain Model

This folder contains the 3D brain model files used in the neural network game portfolio.

## Model Sources
- Brain model: Created using procedural geometry with realistic topology
- Alternative: You can replace with downloaded models from:
  - Sketchfab (free brain models)
  - TurboSquid
  - Free3D
  - NIH 3D Print Exchange (medical models)

## Supported Formats
- .gltf/.glb (recommended)
- .obj
- .fbx
- .dae

## Usage
Place your brain model file in this directory and update the path in main.js
