Quick Implementation

Once you download a model:

  1. Place the file in /public/models/
  2. Rename it to brain.gltf or brain.glb
  3. The system will automatically load it instead of the procedural brain

Supported formats: GLTF, GLB, OBJ, FBX

Model Requirements

  • Scale: Roughly 2-5 units in size
  • Format: GLTF/GLB preferred for best performance
  • Polygons: 10K-50K triangles for good performance
  • Textures: Optional, will be overridden with neural theme
